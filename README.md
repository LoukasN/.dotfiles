# SETTING UP

Set up using the install script (works for arch)
````
cd .dotfiles
./install.sh
````

## Programs list (Hyprland)

- neovim (Editor)
- ly (Login Manager)
- mako (Notification daemon)
- zsh (Shell)
- waybar (Bar)
- papirus-icon-theme (Icon Theme)
- zoxide (Directory )

- slurp (Region capture)
